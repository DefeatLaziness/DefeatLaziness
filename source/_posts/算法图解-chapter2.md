# 选择排序

## 数组

###### 排序是相连的

1. 数组不定长的缺点：在新增元素时，需重新寻找空间
2. 数组定长的缺点：额外多余的位置可能用不上，浪费内存

###### 查找元素：

| 读取 | 插入 | 删除 |
| :--: | :--: | :--: |
| O(1) | O(n) | O(n) |

比链表效率高

## 链表

###### 元素可存储在内存的任何地方，通过指向来连接所有元素

###### 查找元素：

| 读取 | 插入 | 删除 |
| :--: | :--: | :--: |
| O(n) | O(1) | O(1) |
