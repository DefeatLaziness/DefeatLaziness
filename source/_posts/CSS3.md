# CSS3

## 1. 是什么

css, 即层叠样式表(Cascading Style Sheets)的简称，是一种标记语言，由浏览器解析执行用于使页面变得更美观。

Css3 是 css 的最新标准， 是向后兼容的，css1/2的特性在css3里都是可以使用的。

而css3新增了许多新特性，为开发带来了更佳的开发体验。
